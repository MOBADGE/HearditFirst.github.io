**Upcoming Indie Mecha Games to Watch**

With no new FromSoftware Armored Core game expected soon, several indie developers are stepping in to fill the gap for fans of mecha combat. Notable upcoming titles on Steam include:

- *Ironwing Valiant: Record of Astera* by Tiger Team offers a distinct take with verdant landscapes and party customization inspired by Valkyria Chronicles. No release date yet, but it is actively in development with updates available on social media.

- *Code Rapid* from Hitbreak Games features fluid, stylish combat resembling character action games, emphasizing acrobatic moves and reactive rhythms. A demo is planned within a few months.

- *Rig Riot* combines classic Armored Core gameplay with roguelike mechanics, including randomized weapons and permanent progression of mecha parts. A demo is currently available, and the game targets a May 2026 release.

- *Omega Phenex Commenced Project Six* aims for very fast-paced mecha fighting with intense speed reminiscent of Armored Core: For Answer. It is expected in 2027.

These games highlight ongoing interest in mecha action among smaller studios and provide alternatives for players awaiting larger-scale releases.

**Windrose Dedicated Server Setup and Limitations**

Windrose, a pirate-themed survival game, supports dedicated servers for persistent multiplayer worlds. However, the game currently lacks in-game admin commands, limiting server control options. Key points include:

- The dedicated server is Windows-only and has reported issues such as startup crashes and high CPU usage.

- Server configuration is handled by editing JSON files outside the game, allowing adjustments to aspects like server name, password protection, max players, world difficulty, and enemy stats.

- Connecting to the dedicated server requires an invite code found in the server files.

- Community efforts, including Docker containers, face challenges due to technical limitations.

Players intending to host Windrose servers should be prepared for hands-on setup without conventional admin tools. The developers are expected to improve this as the game progresses through early access.

**Guidance on Crafting Gunpowder in Windrose**

Gunpowder in Windrose is essential for firearms but can only be crafted after progressing through specific quests and acquiring necessary tools. The process involves:

- Completing main quests including “Revenge is Best Served Cold” which unlocks access to resources and crafting recipes.

- Obtaining a Millstone, which requires discovering corn and building the crafting station.

- Mining sulfur deposits with an Iron Pickaxe; sulfur appears in both Coastal Jungle and Foothills biomes but resembles stone and can be easy to overlook.

- Collecting ash, usually a byproduct of burning wood at a Charcoal Kiln.

Combining 10 sulfur and 20 ash at the Millstone produces gunpowder. Players unable to craft it should focus on advancing the related quests and equipment requirements first.

**Finding the Hidden Second Chest at Windrose Traveler’s Camps**

Players exploring Windrose’s open world encounter Traveler’s Camps where treasure chests must be located to complete objectives. A common confusion is the seemingly missing second chest at camps marked with two chests. The resolution is:

- Look for a tree nearby with a red scarf tied around it.

- Digging at the base of the marked tree reveals the second chest.

Only some camps provide an explicit note pointing to this clue, but the red scarf method applies widely. This detail helps players fully clear camp locations and gain associated rewards.

**Changes in Subnautica 2’s Publishing and Ongoing Development**

Krafton, initially listed as the publisher for Subnautica 2, is no longer named on the Steam and Xbox store pages; Unknown Worlds is now credited as both developer and publisher. Background:

- Krafton acquired Unknown Worlds in 2021 but fired its leadership in 2025, triggering a lawsuit.

- A judge ordered Krafton to reinstate Unknown Worlds CEO Ted Gill and extended the deadline for Subnautica 2’s early access release to September 15, 2026.

- Krafton states it remains focused on supporting the game’s early access launch, but the publishing change suggests possible shifts behind the scenes.

The early access release date remains unconfirmed. Subnautica 2 continues to be highly anticipated, topping Steam wishlists.
