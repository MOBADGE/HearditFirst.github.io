New Gaming Mouse Comparison Tool Offers Visual Shape Analysis

A website called EloShapes.com has launched a comparison tool for gaming mice and mousepads that emphasizes visual shape and size alongside technical specifications. Traditional reviews often struggle to convey hand-feel, a factor crucial for choosing peripherals, so EloShapes visualizes mice outlines to scale for better size comparison. It also lists key features such as polling rates, sensors, and switches in a clear format. Users can find similar mice based on shape and specs, helping those seeking replacements or alternatives.

While the mouse comparison tool appears useful, the mousepad section is less developed and currently lacks some popular options, including the widely recommended SteelSeries QCK Performance. This tool could be a helpful step toward more tactile-relevant peripheral shopping, but it still has room to grow.

EVE Online Launches AI-Powered Assistance for New Players

MMORPG EVE Online has introduced Aura Guidance, an AI-powered chat assistance feature designed to simplify the game‚Äôs complex mechanics for newcomers. The system leverages over 5.8 million messages from the in-game help channel to distill common questions into concise, context-aware answers. Importantly, CCP Games states Aura Guidance is not generative AI and does not create game content or art but supplements existing player support.

New players will be randomly assigned to groups with or without Aura Guidance to evaluate its impact on retention. The system is opt-in and can be disabled, preserving traditional learning methods for those who prefer them. This aims to make EVE more accessible without replacing community support.

Rocket League to Use Easy Anti-Cheat with Steam Deck and Linux Support

Rocket League announced it will implement Easy Anti-Cheat (EAC) starting in April to improve real-time cheater detection. While EAC disables mod use in online modes, players can still run mods offline with it turned off. Psyonix plans to build popular mod-inspired features directly into the game to offset lost functionality.

Notably, the developers assure players that Steam Deck and Linux versions will remain supported with EAC enabled, unlike some Epic Games titles. Specific tournament-related mods will be available to vetted partners. The update reflects efforts to balance anti-cheat enforcement with user platform support and modding flexibility.
