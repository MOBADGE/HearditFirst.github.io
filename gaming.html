**Arc Raiders Flashpoint Update Introduces New Enemies and Streamlines Crafting**

The third-person shooter Arc Raiders is set to receive its Flashpoint update tomorrow, March 31. This update brings several changes to gameplay, most notably the introduction of a new aerial enemy called the Vaporizer. Equipped with a laser beam, Vaporizers appear in groups and are expected to pose a significant challenge to players. Additionally, Shredders—previously confined to a specific map—will now appear in multiple other locations, increasing the threat level across maps.

To accompany the tougher enemies, the update adds two new weapons: the Dolabra energy shotgun and the Canto sub-machine gun. A new player project, the High-Gain Antenna, brings fresh objectives requiring players to fend off waves of formidable foes while defending a valuable Arc Assessor platform. Smaller but meaningful quality-of-life improvements include a "feeding boost" for the pet companion Scrappy, allowing players to influence resource gathering by feeding specific items, and a streamlined crafting menu that helps players fulfill missing crafting components without leaving the crafting screen.

These adjustments aim to increase challenge variety and reduce some user interface friction, making gameplay more focused and accessible.

**Doom Runs via DNS Text Records in Novel Experiment**

A developer named Adam Rice has demonstrated an unusual way to run the classic shareware version of Doom by splitting its entire game data into 1,964 DNS text records stored within a Cloudflare DNS zone. Using a custom PowerShell script, a player can query all these records to reconstruct and run the game entirely in memory without writing it to disk.

This project highlights how the DNS protocol's text records—originally designed for simple data like email authentication—can store arbitrary content. While DNS was never intended to be a file storage system, this approach exploits its lack of validation. The script reassembles and decompresses the data to launch Doom, albeit with audio stripped to reduce size.

This technical curiosity sheds light on the flexibility and potential abuses of internet infrastructure protocols rather than intending to suggest practical applications.

**Guides for Crimson Desert: Sanctum of Absolution Puzzle and Legendary Mounts**

Several helpful guides have emerged for the RPG Crimson Desert, focusing on how to complete specific in-game objectives and acquire notable mounts.

The Sanctum of Absolution puzzle requires players to defeat the Antumbra’s Sword boss and cleanse the shrine using obtained items such as the Fusion Reactor Core and a battery found by interacting with the environment and using in-game powers. Completing this puzzle unlocks access to a hidden chest containing valuable gear.

For mounts, two significant legendary creatures are highlighted: the Silver Fang wolf and the White Bear. Both mounts require players to engage and defeat the creatures in specified regions and then craft a Sigil of Solidarity using items obtained from the kills at the Witch’s Lair vendor. These mounts not only allow faster traversal but also offer combat abilities to assist in battles.

These guides can help players navigate some of Crimson Desert’s more complex content and enhance their gameplay experience.

**Crimson Desert GPU Driver Issue on ROG Xbox Ally X**

Players attempting to run Crimson Desert on the ROG Xbox Ally X handheld device may face launch problems due to compatibility issues with the latest AMD GPU drivers. Pearl Abyss and Asus recommend rolling back to the previous driver version (V32.0.21025.27003) to resolve the issue while a fix is being developed in collaboration with Microsoft and AMD.

Though inconvenient, this workaround should enable owners of the device to play Crimson Desert until official support catches up. Similar issues have appeared on other hardware platforms, such as Intel Arc GPUs, which Pearl Abyss is also working to support.
