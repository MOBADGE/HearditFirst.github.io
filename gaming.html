**Counterfeit Monkey: A Text Adventure Focused on Language**

*Counterfeit Monkey* is a parser-based text adventure that explores language manipulation as its core gameplay mechanic. Set on the island of Atlantis, the game features a world where the government controls language to suppress public power. Players manipulate words by adding, removing, or changing letters to alter objects, solve puzzles, and progress the story. Unlike typical games that add graphics to puzzles, this game uses text as a primary system, emphasizing cognitive engagement with language. Created by Emily Short, the game has been praised for its intricate worldbuilding, clever narrative integration, and emotional depth. It stands as a noteworthy example of interactive fiction where player input directly interacts with the text-world rather than visual elements.
