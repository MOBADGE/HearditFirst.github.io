**Modder Uses AI to Unlock Obscure Intel Bartlett Lake CPU on Consumer Motherboard**

A modding enthusiast has successfully run an Intel Core 9 273PQE Bartlett Lake CPU—a niche 12 P-core chip designed for embedded and networking use—on a consumer Asus Z790-AYW motherboard. This is notable because Bartlett Lake CPUs were not originally compatible with desktop hardware. The breakthrough reportedly involved rewriting 100% of the motherboard BIOS code using Claude AI, an artificial intelligence tool, marking a rare instance where AI contributed significantly to complex hardware engineering. While the modded BIOS enables Windows to boot with Bartlett Lake, it still has limitations such as inability to access the BIOS without swapping CPUs and incompatibility issues with some standard desktop processors. The achievement highlights AI’s growing role as a practical tool in technical problem-solving, despite some industry skepticism about its reliability.

**New Pretext Library Aims to Improve Web Browsing Performance**

Web developer Cheng Lou, known for work with React and Midjourney, has released a coding library named Pretext designed to dramatically speed up how browsers calculate text layout and positioning. Current web text rendering can be sluggish when dynamically rearranging elements, challenging developers balancing performance and visual accuracy. Pretext accelerates this process significantly, as demonstrated in public demos that handle complex text interactions in real time. Lou used AI to test and refine Pretext across hundreds of browser and language combinations, managing a task that would be otherwise impractical for a single person. While Pretext presently covers limited text scenarios, it is gaining attention for its potential to make websites feel more responsive and reduce lag on text-heavy interfaces.

**Disney Dreamlight Valley Introduces Vault Rewards to Star Path System**

The upcoming April 8 update to Disney Dreamlight Valley will add Vault Rewards to its Star Path progression system. This feature allows players to revisit and earn rewards from previous Star Paths when completing daily routine tasks. If players already own a reward from past paths, the system substitutes it with a random Red Chest to ensure no items are missed. Rewards are granted after every three routine duties, ostensibly making progression feel more continuous and rewarding. However, players cannot select which prior rewards appear, and the rotation details remain unclear. This addition complements the new Pocahontas character and the new Elements of Nature Star Path arriving in the update, suggesting a more layered reward structure for returning players.

**Crimson Desert Patch Updates Graphics and Performance on AMD and Consoles**

Pearl Abyss has released update 1.02.00 for Crimson Desert, which integrates FSR SDK 2.2, promising improved upscaling and frame generation on AMD graphics cards as well as consoles. The patch addresses issues including excessive GPU memory usage with DLSS, flickering when certain graphics options are enabled, and rendering glitches with character faces indoors. Foliage rendering in ray-traced reflections has also been fixed. Despite improvements, some graphical problems related to Nvidia’s DLSS Ray Reconstruction remain unresolved. The developer continues to optimize stability and performance across platforms, reflecting ongoing efforts to polish the game's visual and technical aspects post-launch.

**Team Liquid Secures Fourth Consecutive World of Warcraft Race to World First Title**

After two weeks and 473 raid attempts, Team Liquid has won its fourth straight Race to World First title by defeating the mythic boss L’ura in World of Warcraft: Midnight Season 1. The competition saw intense rivalry with Team Echo, who led at times but ultimately fell short due to a secret fourth phase previously unknown to players. The unexpected final boss phase required adaptation under pressure, with Team Liquid maintaining all 20 players alive for the decisive kill. The event underscores the high skill and dedication involved in competitive raiding. The surprise added narrative complexity to the race, emphasizing how live competitive content can evolve unexpectedly.

**Atlus Surprises Fans with 'Persona 1 Origins' and 'Persona 2 Duality' Rebranding Amid Calls for Remakes**

Atlus has launched merchandise for Persona 1 and Persona 2 under new branding titles—"Persona 1 Origins" and "Persona 2 Duality"—available through Amazon, including apparel and collectibles. While longtime fans have requested remakes or re-releases of the early Persona games, no announcements regarding new game versions have been made. The rebranding has sparked speculation about possible future projects, but currently, it appears limited to a merchandise refresh. Atlus previously expressed interest in remaking the original games but gave no timeline.
