New Gaming Mouse Comparison Tool Aims to Improve Buying Decisions

A new website called EloShapes.com offers a detailed comparison tool for gaming mice and mousepads, focusing on pricing, features, and the physical shape of the peripherals. Unlike typical reviews, the tool uses visual overlays to show the size and silhouette differences between models, helping users better understand how a mouse might fit their hand. It also lists technical features like polling rates, sensor types, and switches for side-by-side evaluation. The site includes a "Find Similar" feature to suggest mice with comparable shapes and specs. However, the mousepad section is still developing and notably lacks some popular options, such as the SteelSeries QCK Performance. This tool could assist gamers in making more informed choices when purchasing peripherals, especially since hand feel is difficult to gauge online.

Rocket League to Implement Easy Anti-Cheat with Mod Support

Rocket League developer Psyonix announced that Easy Anti-Cheat (EAC) will be added to the game starting in April to better detect and ban cheaters. While EAC will disable mods during online play, it can be turned off for offline activities like custom training, replays, and LAN matches. Some community mods, including tournament-specific ones, will be allowed for designated accounts. To address concerns around platform compatibility, Psyonix confirmed that EAC support on Steam Deck and Linux via Proton will be maintained, unlike the controversial Fortnite implementation that blocks these platforms. Additionally, several mod-inspired features are planned to be built directly into the game, reducing the reliance on external mods. This move aims to reduce cheating while balancing community content and platform support.
