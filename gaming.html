**Steam Workshop Adds Version Control to Ease Mod Compatibility**

Valve has introduced version control features to Steam Workshop to help address mod compatibility issues following game updates. Mod authors can now upload multiple versions of their mods, each marked with compatibility information for specific game versions. This should allow players to select mod versions that match their installed game version more easily, reducing frustration caused by updates breaking mods and related save files.

The new API also enables developers to detect mismatches between game and mod versions and offer prompts or automatic adjustments to ensure smoother play. This functionality depends on adoption and proper configuration by both game developers and mod creators but promises to improve the overall modding experience on Steam.
