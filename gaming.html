Palia's Upcoming Expansion Adds Realistic, Breedable Horses

Palia, the life simulation MMO, will introduce its next expansion, The Royal Highlands, on May 12. A key highlight is the addition of breedable and rideable horses, a feature frequently requested by MMORPG players. Unlike typical mounts in other games that mainly serve as speed boosts, Palia’s horses have multiple gaits, traits affecting their handling, and a breeding system to pass favored characteristics to offspring. Three fantasy-inspired horse breeds will be available at launch. The update also includes a new adventure zone, new furniture, ranches for home lots, and a new villager. This expansion aims to deepen player interaction with the game world and enhance the ranching aspect.

Hunch: A More Challenging Wordle-Like Puzzle

For players seeking a tougher alternative to Wordle, the daily puzzle game Hunch offers a more complex experience. While it employs the familiar five-letter guessing format and color-coded feedback, Hunch differs by only revealing the number of correct letters in the correct or wrong positions, without specifying which ones. This mechanic, reminiscent of Mastermind, means players may require many more than six guesses to solve a puzzle. Hunch includes a daily challenge, a "cascade" mode with consecutive puzzles, and multiplayer competitions. It presents a more demanding word puzzle for players interested in a longer-term mental challenge.

Framework Announces New Wireless Keyboard and DIY Support

Hardware manufacturer Framework, known for its modular laptops, unveiled a new wireless keyboard featuring a built-in trackpad designed for living room or simulator rig setups. The keyboard uses chiclet-style keys and a Windows Precision Touchpad compatible with Windows and Linux gesture controls. Framework also released CAD files enabling users to build custom mounts for the keyboard and announced the Control Board—its core Bluetooth and USB connectivity component—will be sold separately for custom wireless keyboard projects. The keyboard aims to offer a versatile and modular input solution, with availability expected later this year.

Rust Player Develops 'Horse Baiting' Tactic to Counter Horse Shield Meta

In the survival game Rust, a problematic "horse shield" tactic—riding a horse equipped with a shield to gain an overwhelming combat advantage—continues to frustrate players, despite developer attempts at fixes on test servers. A Rust YouTuber known as Fyx created a counter-strategy using horses as bait, luring players toward traps like landmines, shotgun rigs, and C4 explosives. These setups exploit players’ eagerness to capture horses, thereby turning the cheap meta tactic against them. This player-driven innovation highlights the ongoing emergence of community strategies to address balance issues while official patches remain in progress.

Xbox Game Pass Reduces Subscription Prices but Delays Call of Duty Additions

Microsoft has announced price reductions for its Xbox Game Pass subscription: the Ultimate tier drops from $29.99 to $22.99 per month, and PC Game Pass falls from $16.49 to $13.99. Despite the cuts, this does not revert the service to previous pre-hike pricing levels. In tandem, Microsoft revealed future Call of Duty titles will no longer be available on Game Pass at launch. Instead, they will be added approximately a year later, beginning with the next release in the series. This change follows reports that Game Pass availability led to reduced full-price game sales, suggesting a shift in Microsoft's approach to balancing subscription value with game revenue.

Framework Unveils Framework 13 Pro Laptop with Enhanced Upgradability and Battery Life

Framework announced the Framework 13 Pro, positioned as a "MacBook Pro for Linux users," launching in June. The model features a fully machined aluminum chassis, a new Panther Lake mainboard with support for the latest LPCAMM2 memory and PCIe Gen5 SSDs, and a larger 74 Wh battery. It maintains full compatibility with prior Framework 13 components for easy upgrades or replacements. The new 13 Pro includes a custom 2880x1920 3:2 display with 700 nits brightness and variable refresh rates, plus a new haptic trackpad designed to improve input feel under Windows and Linux. Prebuilt configurations start at $1,499, with a DIY kit available at $1,199. This release continues Framework’s focus on sustainability and user repairability.
