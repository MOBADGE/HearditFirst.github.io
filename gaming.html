Microsoft Marks 10 Years of DirectX 12 with Ongoing Evolution

Microsoft recently celebrated a decade since the release of DirectX 12, a key low-level graphics API that launched alongside Windows 10 in 2015. While Windows 10 itself has moved towards end-of-life status, DirectX 12 has continued to evolve, becoming one of the most comprehensive collections of graphics APIs available. Over the years, Microsoft has added features such as ray tracing (DXR), variable rate shading, mesh shaders, and DirectStorage, though some additions like DirectStorage have seen limited adoption so far.

DirectX 12 has adapted to the changing hardware landscape where GPUs now rely more on compute shaders than traditional vertex and pixel shaders. Recent updates introduced features like Work Graphs, Shader Execution Reordering, and Cooperative Vectors to better leverage modern GPU architectures. Despite the many updates, Microsoft has not announced a DirectX 13, opting to improve the current API iteratively rather than through a full new version. This approach reflects the industry's mixed needs, as some games like Baldur‚Äôs Gate 3 still use DirectX 11 or Vulkan without performance penalties.
