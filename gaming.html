**SK Hynix Speeds Up Chip Production to Meet AI Demand**

South Korean memory manufacturer SK Hynix has announced that it will complete the first fab at its new Yongin Semiconductor Cluster three months earlier than planned, now targeting February 2027. Additionally, an existing fab in Cheongju (M15X) will begin producing silicon wafers for high-bandwidth memory (HBM) chips next month. These moves respond to substantial and growing demand for memory chips in AI data centers. While this expansion will primarily benefit large-scale infrastructure customers, it could gradually ease the overall memory supply shortage and impact prices. **Wine 11 Release Brings Improvements to Linux Gaming and SteamOS**

Wine, the compatibility layer that enables Windows software to run on Linux, has reached version 11.0, introducing several updates that improve gaming performance on Linux distributions and SteamOS devices like the Steam Deck. Key improvements include full support for WoW64 mode (allowing 32-bit and 16-bit Windows applications), kernel module NTsync support which reduces latency for demanding games, and enhancements to Wayland protocol support such as clipboard integration and input methods. The release also adds exclusive fullscreen mode support and various graphics fixes. Since Valve's Proton builds on Wine, these updates will indirectly enhance the experience of Windows games running on Linux-based gaming systems. This continues a trend of yearly Wine updates improving compatibility and performance on non-Windows platforms.
