How to Use Emotes in Nioh 3

Nioh 3 offers a nuanced emote system referred to as "gestures," which players can access by pressing the right bumper and start button on a controller, or the 'G' key on a keyboard. This opens a gesture menu with several pages of emotes. Beyond simple player communication or photo opportunities, emotes have gameplay significance against two yokai types: Nurikabe and Mujina. Nurikabe, disguised as walls, react differently depending on the color-coded emote used (red, yellow, or blue), allowing players to make these yokai disappear if the correct color is matched. Mujina, which mimic chests and drop loot, mimic the player‚Äôs emote upon chest opening and must be replied to with the same emote to avoid combat. Understanding this adds a layer of strategy and utility to what might otherwise be considered a cosmetic feature.

Steam Proton Fix Improves Linux Gaming Experience

A recent Steam beta update has addressed a significant issue with Proton, the compatibility layer that allows Windows-only games to run on Linux. Previously, users with very large game libraries experienced a bug where Proton games would incorrectly show as "Not valid on current platform," effectively preventing them from launching. The fix restores the core function of Proton for these users. Additional improvements included in the beta target client download views, controller LED behavior, and button configuration settings. This update reflects ongoing efforts to enhance Linux gaming support, making it more accessible for players adopting Linux as their gaming platform.
