Memory Module Manufacturer Patriot Addresses Ongoing RAM Price Crisis

At CES 2026, Patriot's marketing manager Shannon Robb spoke candidly about the current memory supply challenges. The RAM market continues to face high prices largely due to AI companies consuming approximately 40% of wafer production before chips even reach memory module manufacturers like Patriot. Much of this stock is held in storage as data centers struggle with the power required to bring it online. Robb emphasized that Patriot is not benefiting financially from the price surge but is forced to absorb higher costs for integrated circuits (ICs), which primarily come from major manufacturers such as Samsung and SK Hynix.

While other companies like Micron are retreating from the consumer memory market to focus on data centers, Patriot plans to stay committed to PC gamers despite expecting prices to rise further through 2026. Robb advised consumers to prioritize practical memory upgrades based on their needs rather than chasing top-tier kits, suggesting that if current hardware plays games adequately, delaying upgrades is reasonable amid uncertain pricing trends.

Windows Secure Boot Certificates for Anti-Cheat to Expire in June, Updated Certificates Rolling Out

Microsoft announced that Secure Boot certificates, which verify firmware authenticity during system startup, are set to expire in June 2026. Secure Boot plays a significant role in underpinning anti-cheat systems used by major game developers including EA, Riot, and Activision. To address this, Microsoft is distributing updated certificates (CA 2023) via Windows Updates to PCs that still rely on the older CA 2011 certificates.

Most newer machines already contain the updated certificates, but systems without them could experience issues with Secure Boot verification after expiration. Users can check their current certificates using Windows PowerShell commands. While the update process is largely automatic, maintaining current Windows updates is recommended to ensure anti-cheat functionalities remain intact.
