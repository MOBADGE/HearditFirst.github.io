Valve Testing Frame Rate Estimates on Steam

Valve is developing a new feature for Steam that would provide players with estimated frame rates for games on their own PC hardware. This comes from data collected via an opt-in system where users send anonymized frame rate information to Valve. The system would allow players to input their CPU, GPU, and RAM specifications—or have Steam detect them automatically—and then receive an estimate of how well a game will perform. The accuracy depends on the volume of data Valve collects, likely restricting estimations to more popular titles. This update matters because it could help buyers better gauge game performance before purchase, potentially reducing frustration over poor optimization. Unlike third-party tools, Valve’s approach promises broad coverage and integration within Steam.

Framework Warns Memory Price Stability is Temporary

Modular laptop maker Framework has cautioned that recent stabilization in memory prices is only a short break amid ongoing volatility. CEO Nirav Patel noted that while some areas show stable costs, memory and SSD prices are expected to rise further through 2026. Framework has kept DDR5 RAM prices steady for now and is using older, cheaper SSD inventory, but new stock comes at higher costs, prompting price hikes on specific products like 4TB WD SN850x SSDs. Framework’s desktop memory pricing saw increases, especially for high-capacity LPDDR5x modules, although certain laptop configurations benefited from reduced CPU prices. This update is important as it underscores continued supply-chain pressures affecting PC hardware costs, driven by AI-related demand and broader market uncertainty.

Overwatch Updates Anran’s Character Model Ahead of Season 2

Blizzard has released a redesign of Anran, a new hero in Overwatch, following criticism of her initial in-game appearance for looking too young and too innocent. The updated model adjusts her facial features and posture to give her a more mature, confident look that better reflects her intended characterization as Wuyang’s older sister. This redesign was welcomed by Anran’s voice actress and many players, though some still note visual differences compared to earlier comic depictions. Blizzard also indicated that the process of refining Anran’s design helped inform work on upcoming heroes, suggesting ongoing adjustments may occur for future additions. The revision highlights the impact of community feedback on character presentation in live-service games.

Sentinel Adds Achievements to Non-Steam Linux Games

A new open-source tool called Sentinel allows Linux gamers to add achievement tracking to games played outside Steam, including titles launched through other storefronts like Epic Games. Using Steam’s Web API or external databases, Sentinel monitors game saves and awards achievements in real time with notifications similar to Steam’s system. It supports multi-step achievements and tracks overall progress, enhancing the experience for players who value accomplishment markers but play outside Steam. As achievement hunting is a niche but enthusiastic segment of the gaming community, this tool fills a gap by extending achievement functionality to a broader set of games on Linux. Future plans include compatibility with Steam Deck devices.

Pillars of Eternity Adds Turn-Based Combat Mode

Obsidian has released a significant update for Pillars of Eternity, adding a full turn-based combat mode nearly 11 years after the game’s original launch. This mode differs from the traditional real-time-with-pause system by limiting players to one action per turn but allowing multiple turns per round based on speed and recovery stats. It provides greater tactical depth and changes class balance, benefiting certain builds like Ciphers. The new system simplifies timing but emphasizes strategic choices and crowd control. Early impressions suggest the turn-based mode may offer a more accessible and refreshed gameplay experience, appealing to both newcomers and longtime fans. This addition reflects a broader trend of revisiting classic RPG mechanics in modern updates.

Linux Kernel Dropping Support for Intel 486 CPUs

The Linux kernel maintainers are beginning to remove support for Intel’s older 486 processors, nearly three decades after the last desktop 486 chips were produced. The 486 architecture debuted in 1989 and was officially discontinued for desktops in 1998, though some embedded versions persisted until 2007. Kernel lead Linus Torvalds considers 486-class hardware obsolete and irrelevant to current development. The removal helps clean up legacy compatibility code that complicates maintenance and causes occasional issues. While some enthusiasts may still run Linux on vintage 486 machines using older kernels, mainstream distributions no longer support such aging hardware. Support for later Pentium-era processors will continue as those remain more relevant.
